1. What is CSRF and Why It Matters

Cross-Site Request Forgery (CSRF) is an attack where a malicious website tricks a user's browser into making unintended requests to another site where the user is authenticated. If the target site relies on cookies, the browser automatically includes them, allowing the attacker to act as the user.

Why CSRF Protection Even with JWT?

  • If a future change introduces cookie-based sessions, CSRF protection is already in place
  • Prevents token relay attacks where an attacker's page could potentially extract and replay tokens
  • Satisfies security audit requirements for financial/e-commerce applications
  • Validates that requests originate from our UI, not any client with a valid JWT

Request Flow

User Action → POST/PUT/DELETE with X-CSRF-Token header → CsrfValidationFilter

3.1 Files Created

CsrfTokenService.java

ecom-service/src/main/java/com/bookstore/ecom/config/CsrfTokenService.java

@Service
public class CsrfTokenService {
    private static final String KEY_PREFIX = "csrf:";
    private static final Duration TOKEN_TTL = Duration.ofMinutes(30);

    public String generateToken(String userId) {
        String token = UUID.randomUUID().toString();
        redisTemplate.opsForValue().set(KEY_PREFIX + userId, token, TOKEN_TTL);
        return token;
    }

    public boolean validateAndRefresh(String userId, String token) {
        if (token == null || token.isBlank()) return false;
        try {
            String stored = redisTemplate.opsForValue().get(KEY_PREFIX + userId);
            if (stored != null && stored.equals(token)) {
                redisTemplate.expire(KEY_PREFIX + userId, TOKEN_TTL);
                return true;
            }
            return false;
        } catch (Exception e) {
            log.warn("Redis error -- failing open: {}", e.getMessage());
            return true;  // JWT is the primary defense
        }
    }
}

CsrfTokenController.java

ecom-service/src/main/java/com/bookstore/ecom/controller/CsrfTokenController.java

@RestController
public class CsrfTokenController {
    @GetMapping("/csrf-token")
    public Map<String, String> getCsrfToken(JwtAuthenticationToken auth) {
        String userId = auth.getToken().getSubject();
        String token = csrfTokenService.generateToken(userId);
        return Map.of("token", token);
    }
}
